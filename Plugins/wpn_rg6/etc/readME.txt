[en]
If you want to custom kill icons when someone is being killed with this grenade launcher:

1) go to your sprites folder
2) put there hud_desert.spr
3) open hud.txt
4) add somewhere this
	d_gp25_grenade		640 	hud_desert 170 	45 	64 	16
5) if game is currently running - restart it

[ru]
Если вы хотите видеть иконку при фраге с этой пушки

1) Откройте папку sprites
2) киньте туда hud_desert.spr
3) Откройте hud.txt
4) Добавьте в него куда нибудь это
	d_gp25_grenade		640 	hud_desert 170 	45 	64 	16
5) если игра запущена - сделайте рестарт.
